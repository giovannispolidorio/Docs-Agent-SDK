# Autenticação

Para que o SDK seja utilizado é necessaria solocitar a Olos a autenticação/autorização do SDK. Para solicitar as chaves de SDK abrir um ticket para o nosso time de [Professional Service (OPS)](https://support.olos.com.br/a/catalog/request-items?search=ops). Após solicitado o time de OPS vai fornecer o **clientID** e **clientSecret**
