# Eventos

O Agent SDK é assíncrono, desta forma, para cada método disparado retornará um ou mais eventos e por consequência esses métodos não terão nenhum retorno,
assim que a inicialização for feita o SDK deverá começar a disparar os eventos do Olos para o Agente. Nessa sessão vamos demonstrar como a sua aplicação pode escutar ou ignorar esses eventos e a descrições
da funcionalidade de cada evento.

## OlosOn

Para sua aplicação conseguir escutar os eventos do Olos Agent SDK vai ser necessário `importar` a função do SDK chamado _olosOn_.

```js copy
import { olosOn } from "olosagentsdk";

olosOn("<nomeDoEvento>", callback);
```

## OlosOff

Através do _olosOff_ o SDK entende que a aplicação não quer mais ouvir o evento.

```js copy
import { olosOff } from "olosagentsdk";

olosOff("<nomeDoEvento>", callback);
```
