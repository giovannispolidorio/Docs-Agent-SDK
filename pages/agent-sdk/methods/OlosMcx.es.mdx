# OlosMcx

OlosMcx le permite controlar y administrar la jornada del cliente en la plataforma Olos.

## getJourney

Solicita la creación de un identificador de Jornada del cliente.

```ts copy
import { OlosMcx } from "olosagentsdk";

const mcx = new OlosMcx();
mcx.getJourney(
    timeout: number | null,
    callbackSuccess: (response: McxResponse) => void,
    callbackError: () => void
);
```

## disposition

Solicita que se tabule una llamada de voz.

```ts copy
import { OlosMcx } from "olosagentsdk";

const mcx = new OlosMcx();
mcx.disposition(
    information: unknown,
    timeout: number | null,
    callbackSuccess: (response: McxResponse) => void,
    callbackError: (err: any) => void
);
```
