# OlosAgentWS

OlosAgentWS contiene todos los métodos de comunicación que permiten controlar y gestionar el estado de los agentes y llamadas conectadas al marcador.

## agentAuthentication

Autenticación de agentes en la plataforma Olos en el Marcador y Campañas de Texto.

```ts copy
import { OlosAgentWS } from "olosagentsdk";

const agentWS = new OlosAgentWS();
agentWS.agentAuthentication(login: string, passwd: string, callback?: (agentId: string) => void);
```

## agentAuthenticationAzure

Autenticar el agente en la plataforma Olos en el Marcador y Campañas de Texto para integraciones con el proveedor de identidad de Microsoft Azure Active Directory.

```ts copy
import { OlosAgentWS } from "olosagentsdk";

const agentWS = new OlosAgentWS();
agentWS.agentAuthenticationAzure(tokenId: string, callback?: (agentId: string) => void, callbackError?: (msgError: string) => void);
```
