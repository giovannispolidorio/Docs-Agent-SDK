# OlosAgentChannelWs

The OlosAgentChannelWs contains all communication methods that allow control and management of the state of agents in Text Campaigns.

## agentAuthenticationChat

Agent authentication on the Olos platform, specific to Text Campaigns.

```js copy
import { OlosAgentChannelWs } from "olosagentsdk";

const agentChannel = new OlosAgentChannelWs();
agentChannel.agentAuthenticationChat("login", "password", callback);
```

## agentAuthenticationAzureChat

Authenticate the Agent on the Olos platform specifically for Text Campaigns for integrations with the Microsoft Azure Active Directory identity provider.

```js copy
import { OlosAgentChannelWs } from "olosagentsdk";

const agentChannel = new OlosAgentChannelWs();
agentChannel.agentAuthenticationAzureChat("tokenId", callback);
```

## dispositionChat

  hour: string;
  minute: string;
  privateAgent: string;
};

## transferChat

Requests the transfer of a conversation from the text campaign to another campaign or Agent.

```ts copy
import { OlosAgentChannelWs } from "olosagentsdk";

const agentChannel = new OlosAgentChannelWs();
agentChannel.transferChat(
    dialogId: string,
    dispositionId: string,
    dispositionCode: string,
    campaignIdDest: string,
    agentIdDest: string,
    year: string,
    month: string,
    day: string,
    hour: string,
    minute: string,
    privateTransfer: boolean
);
```
