# OlosAgentWS

OlosAgentWS contains all communication methods that allow control and management of the state of agents and connected calls in the dialer.

## agentAuthentication

Agent authentication on the Olos platform for the Dialer and Text Campaigns.

```ts copy
import { OlosAgentWS } from "olosagentsdk";

const agentWS = new OlosAgentWS();
agentWS.agentAuthentication(login: string, passwd: string, callback?: (agentId: string) => void);
```

## agentAuthenticationAzure

Authenticate the Agent on the Olos platform for the Dialer and Text Campaigns for integrations with the Microsoft Azure Active Directory identity provider.

```ts copy
import { OlosAgentWS } from "olosagentsdk";
```
