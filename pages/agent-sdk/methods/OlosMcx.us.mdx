# OlosMcx

The OlosMcx allows control and management of the customer journey on the Olos platform.

## getJourney

Requests the creation of a journey ID for the customer.

```ts copy
import { OlosMcx } from "olosagentsdk";

const mcx = new OlosMcx();
mcx.getJourney(
    timeout: number | null,
    callbackSuccess: (response: McxResponse) => void,
    callbackError: () => void
);
```

## disposition
