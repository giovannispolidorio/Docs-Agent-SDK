# Métodos

A través de los métodos de las APIs es posible enviar 'comandos' para comunicarse con la plataforma Olos y para facilitar separamos los métodos en algunos objetos 
haciendo necesario hacer la 'importación' solo de los objetos que contienen los métodos necesarios para la integración.

### Lista de objetos

- OlosAgent
- OlosAgentChannelWs
- OlosAgentWS
- OlosIntegrationWs
- OlosMailingWS
- OlosMcx
- OlosRecordingRetrieve
- OlosSoftPhone
- OlosUserConfigWS
- OlosVoiceSupportWS
