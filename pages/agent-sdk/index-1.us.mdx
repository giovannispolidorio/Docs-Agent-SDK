# Introduction

Welcome to the Olos Agent SDK! Through this documentation, you will learn how to communicate with the Olos platform using methods and events in a customer service operation.

## What is Agent SDK?

The Agent SDK is a Vanilla JS and NodeJs library provided by Olos, enabling the control and management of the state of agents and connected calls in OLOS. It notifies the application about events occurring in the Dialer and Text Channels. This document will present details of the interface, its properties, methods, events, and instructions for its correct usage.
