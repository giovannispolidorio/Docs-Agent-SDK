# Métodos

Através dos métodos das APIs é possível enviar `comandos` para se comunicar com a plataforma Olos e para facilitar separamos os métodos em alguns objetos
fazendo com que seja necessário fazer o `import` apenas dos objetos que contém os métodos necessários para a integração.

### Lista de objetos

- OlosAgent
- OlosAgentChannelWs
- OlosAgentWS
- OlosIntegrationWs
- OlosMailingWS
- OlosMcx
- OlosRecordingRetrieve
- OlosSoftPhone
- OlosUserConfigWS
- OlosVoiceSupportWS
