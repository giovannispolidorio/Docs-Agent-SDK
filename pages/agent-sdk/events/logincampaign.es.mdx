# Logincampaign

Evento exitoso en el inicio de sesión del agente en una campaña

## Como escuchar

```js copy
import { olosOn } from "olosagentsdk";

olosOn("LoginCampaign", callback);
```

## Payload

```json copy
{
  "agentId": 108,
  "campaignId": 2,
  "campaignName": "Receptivo1",
  "campaignCode": "Campaña Receptiva 1"
}
```
