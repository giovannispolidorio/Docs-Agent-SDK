# Logincampaign

Event of success in agent login for a campaign.

## How to listen?

```js copy
import { olosOn } from "olosagentsdk";

olosOn("LoginCampaign", callback);
```

## Payload

```json copy
{
  "agentId": 108,
  "campaignId": 2,
  "campaignName": "Inbound1",
  "campaignCode": "Inbound Campaign 1"
}
```
