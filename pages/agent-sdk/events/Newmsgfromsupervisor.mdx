# Newmsgfromsupervisor

## Descrição

Este evento é o evento retorna a mensagem enviado pelo supervisor para o operador.

## Como escutar

```js copy
import { olosOn } from "olosagentsdk";

olosOn("NewMsgFromSupervisor", callback);
```

## Payload

```json copy
{ "messageId": 90046, "broadcastId": 0, "agentId": 48, "supervisorId": 49, "message": "Mensagem para o Operador" }
```
