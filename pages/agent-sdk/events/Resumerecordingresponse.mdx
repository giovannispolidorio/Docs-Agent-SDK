# Resumerecordingresponse

## Descrição

Este evento é retorno do método resumeRecording com o resultado da solicitação.

## Como escutar

```js copy
import { olosOn } from "olosagentsdk";

olosOn("ResumeRecordingResponse", callback);
```

## Payload

```json copy
{ "result": "true", "code": "0", "description": "" }
```
