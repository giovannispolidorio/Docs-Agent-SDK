# Sendmsgtosupervisorfail

Este evento es disparado cuando hay un error enviando el mensaje al supervisor con la informaci√≥n sobre los errores.

## Como escuchar

```js copy
import { olosOn } from "olosagentsdk";

olosOn("SendMsgToSupervisorFail", callback);
```

## Payload

```json copy
{
  "agentId": 48,
  "reason": 2,
  "description": "SupervisorId 1584 not found or is not a supervisor or is not activated."
}
```

## Errores posibles

1. GenericError
2. AgentIdNotFound
3. SupervisorNotFound
4. ServiceNotAvail
5. SysConfNotFound
6. NotSupervisorOfAgent,
7. EmptyMessage
