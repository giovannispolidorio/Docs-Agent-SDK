# Pauserecordingrespons

This event is a return from the pauseRecording method with the result of the request.

## How to listen?

```js copy
import { olosOn } from "olosagentsdk";

olosOn("PauseRecordingRespons", callback);
```

## Payload

```json copy
{ "result": "true", "code": "0", "description": "" }
```
