# Logincampaignfail

Evento de error de inicio de sesión del agente en una campaña.

## Como escutar

```js copy
import { olosOn } from "olosagentsdk";

olosOn("LoginCampaignFail", callback);
```

## Payload

```json copy
{
  "agentId": 108,
  "campaignId": 2,
  "campaignName": "Receptivo1",
  "campaignCode": "Campaña Receptiva 1"
}
```
