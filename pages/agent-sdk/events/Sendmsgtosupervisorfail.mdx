# Sendmsgtosupervisorfail

## Descrição

Este evento é disparado quando houver um erro no envio da mensagem para o supervisor com as informaçõ es sobre os erros.

## Como escutar

```js copy
import { olosOn } from "olosagentsdk";

olosOn("SendMsgToSupervisorFail", callback);
```

## Payload

```json copy
{ "agentId": 48, "reason": 2, "description": "SupervisorId 1584 not found or is not a supervisor or is not activated." }
```

## Erros possíveis

1. GenericError
2. AgentIdNotFound
3. SupervisorNotFound
4. ServiceNotAvail
5. SysConfNotFound
6. NotSupervisorOfAgent,
7. EmptyMessage
