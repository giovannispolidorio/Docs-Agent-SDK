# Refercallrequestfail

## Como escuchar

```js copy
import { olosOn } from "olosagentsdk";

olosOn("ReferCallRequestFail", callback);
```

Error en la solicitud ReferCall _changestatus_. En caso de éxito el Agente recibirá un evento de estado con el resultado de Wrap, informando que ha sido liberado de la llamada y puede tabular.
