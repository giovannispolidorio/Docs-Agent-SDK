# Logincampaign

## Descrição

Evento de sucesso no login do agente em uma campanha.

## Como escutar

```js copy
import { olosOn } from "olosagentsdk";

olosOn("LoginCampaign", callback);
```

## Payload

```json copy
{ "agentId": 108, "campaignId": 2, "campaignName": "Receptivo1", "campaignCode": "Campanha Receptiva 1" }
```
