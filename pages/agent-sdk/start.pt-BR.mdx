# Arquitetura

O Agent SDK se comunica com a plataforma Olos através de APIs Rest com autenticação Oauth2, para que seja efetuado testes de integrações
será necessário um provionamento prévio no Ambiente de Testes da plataforma Olos.

## APIs e Credenciais

Solicite os endereços das APIs como também as credenciais de autenticação para o nosso time de Professional Service (OPS),
entre em contato com o seu gerente de contas na Olos.

**Após solicitado o time de OPS vai fornecer os endereços das seguintes APIs:**

- WebAPIAgentControl
- WebApiConfiguration
- WebApiMailingControl
- WebAPISoftphone
- WebApiRecordingRetrieve
- wsSoftphone
- WsMcx

**Credenciais de Usuário:**

- Usuário de integração
- Senha do usuário de integração

**Credenciais de Aplicação**

- clientID
- clientSecret
