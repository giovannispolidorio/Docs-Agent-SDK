# Architecture

The Agent SDK communicates with the Olos platform through Rest APIs with OAuth2 authentication. To perform integration tests, prior provisioning in the Olos platform's Test Environment is required.

## APIs and Credentials

Request the API addresses as well as the authentication credentials for our Professional Service (OPS) team by clicking
[here](https://support.olos.com.br/a/catalog/request-items?search=ops).

**After the request, the OPS team will provide the addresses for the following APIs:**

- WebAPIAgentControl
- WebApiConfiguration
- WebApiMailingControl
- WebAPISoftphone
- WebApiRecordingRetrieve
- wsSoftphone
- WsMcx

**User Credentials:**

- Integration user
- Integration user's password

**Application Credentials**

- clientID
- clientSecret
