# Arquitectura

El SDK del agente se comunica con la plataforma Olos a través de API Rest con autenticación Oauth2, para las pruebas de integración 
se requerirá un provisionamiento previo en el entorno de laboratorio de la plataforma Olos.

## APIs y Credenciales

Solicite las direcciones de las API, así como las credenciales de autenticación para nuestro equipo de Professional Service (OPS) haciendo clic
[aqui](https://support.olos.com.br/a/catalog/request-items?search=ops).

**Una vez solicitada, el equipo de OPS proporcionará las direcciones de las siguientes APIs:**

- WebAPIAgentControl
- WebApiConfiguration
- WebApiMailingControl
- WebAPISoftphone
- WebApiRecordingRetrieve
- wsSoftphone
- WsMcx

**Credenciales de Usuario:**

- Usuario de integración
- Contraseña de usuario de integración

**Credenciales de Aplicación**

- clientID
- clientSecret
