# Instalação

Após receber as devidas autorizações instalar o SDK usando o _NPM_

```js copy
npm i @olostecnologia/olosagentsdk
```

# Inicialização

Após a Instalação ser concluida é necessario fazer a inicialização do sdk utilizando a função _createOlos_

```js copy
import { createOlos } from "olosagentsdk";

const addrs = {
  wsAgentCmd: "<server>:<porta>/WebAPIAgentControl_Token/AgentCommand/",
  wsAgentEvt: "<server>:<porta>/WebAPIAgentControl_Token/AgentEvent/",
  WsAgentCloud: "<server>:<porta>/WebAPIAgentControl_Token/CloudAgent/",
  wsMailingCmd: "<server>:<porta>/WebAPIMailingControl_Token/MailingCommand/",
  wsAgentConfig: "<server>:<porta>/WebAPIConfiguration_Token/AgentConfig/",
  wsVoiceSupport:
    "<server>:<porta>/WsVoiceSupportIntegration/WsVoiceSupportIntegration.asmx",
  WebAPIAgentControl: `<server>:<porta>/WebAPIAgentControl_Token/`,
  wsSoftphone: `<server>:<porta>/WebAPISoftphone_Token/Softphone/`,
  wsMcx: `<server>:<porta>/WsMcx_Token/wsmcx/Mcx/`,
  wsRecordingRetrieve: `<server>:<porta>/WebApiRecordingRetrieve_Token/RecordTextComm/`,
};

const auth = {
  user: "agente",
  password: "123",
  clientID: "<clientID>",
  clientSecret: "<clientSecret>",
};

const jsLogger = false;

createOlos(addrs, auth, jsLogger);
```

## Parametros para inicializar

Para podermos inicializar o SDK é necessario passar três paramentros para a função _createOlos_

### addrs

Nesse parametro devemos informar para o SDK quais os servidores que as APIs devem ser apontadas

### auth

Nesse parametro devemos informar para o SDK quais as informações para autenticar a API

```json copy
{
  "user": "Nome do Agente",
  "password": "Senha do Agente",
  "clientID": "clientID fornecido pela Olos",
  "clientSecret": "clientSecret fornecido pela Olos"
}
```

### jsLogger

Nesse parametro informamos se o SDK de inicializar com a função de logs ligada
